Into The Keep - Demo

Controls

  • A to move left
  • D to move right
  • W to jump
This demo is a brief proof of concenpt of a javascript game rendered with HTML5 canvas. It has been made in 3 weeks, and is still under development.
Each tile is 64 pixels by 64 pixels and is cut from this tile sheet. Using a tile sheet like this allows for a faster work flow and a smaller load from asset requests on the program. The demo is currently 100 tiles wide and 10 tiles high. The full game will be much larger than this, with enemies and bosses to fight. The full game will be a 12 week project that I will only be able to work on in my free time.

This demo displays object oriented programming with javascript and use of the HTML canvas API.
